1. Proje Tanımı

Ehliyet AI, kullanıcıların ehliyet teorik sınavına hazırlanmasını kolaylaştıran bir mobil uygulamadır. Uygulama, yapay zeka destekli soru çözme, deneme sınavları ve kişiye özel öğrenme sistemi ile kullanıcıların eksik konularını analiz ederek daha verimli bir çalışma deneyimi sunmayı amaçlamaktadır.

Uygulama React Native kullanılarak geliştirilecektir ve hem Android hem iOS platformlarında çalışacaktır.


2. Projenin Amacı

Bu projenin temel amacı:
	•	Ehliyet sınavına hazırlanan kullanıcıların kolay şekilde çalışabilmesini sağlamak
	•	Kullanıcıların deneme sınavları çözmesini sağlamak
	•	Yapay zeka ile yanlış yapılan konuları analiz etmek
	•	Kullanıcılara kişiselleştirilmiş soru önerileri sunmak

⸻

3. Kullanılan Teknolojiler

Mobil Uygulama
	•	React Native
	•	Expo veya React Native CLI
	•	JavaScript / TypeScript

Backend (Opsiyonel)
	•	Node.js
	•	Express.js
	•	REST API

Veritabanı
	•	Firebase veya MongoDB

Yapay Zeka
	•	OpenAI API veya benzeri LLM
	•	Kullanıcı performans analizi

⸻








4. Sistem Mimarisi

Sistem üç ana bileşenden oluşmaktadır:

1) Mobil Uygulama

React Native ile geliştirilecek ve kullanıcı arayüzünü sağlayacaktır.

2) Backend Sunucu

Kullanıcı verileri, soru bankası ve analiz işlemleri burada yapılacaktır.

3) AI Servisi

Kullanıcıların çözmüş olduğu sorular analiz edilerek:
	•	zayıf konular
	•	önerilen sorular
	•	çalışma planı

oluşturulacaktır.

⸻
